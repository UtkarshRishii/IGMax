## 🌟 Overview

**FalMax Carousel Agent** is an autonomous AI system designed for creators, marketers, and entrepreneurs. It takes a simple topic and transforms it into a complete, professional-grade Instagram carousel. 

Unlike simple text generators, FalMax handles the **Strategy**, **Copywriting**, and **Graphic Design** automatically, outputting high-fidelity 1080x1080 PNG slides ready for upload.

## ✨ Features

- 🧠 **Smart Content Strategy**: Powered by Llama 3 on Groq for "Knowledge Bomb" style content.
- 🎨 **Premium Templates**: Custom-coded HTML/CSS designs with futuristic aesthetics.
- 📸 **High-Fidelity Rendering**: Automated 1080x1080 PNG export via Playwright.
- ⚡ **Lightning Fast**: Optimized rendering engine with browser instance pooling.
- 📦 **One-Click Export**: Downloads everything (Slides + Captions + Hashtags) in a single ZIP bundle.
- 📱 **Modern Dashboard**: A sleek, glassmorphic web UI for seamless creation.

## 🛠️ Tech Stack

- **Backend**: FastAPI (Python)
- **AI Layer**: Groq Cloud API (Llama 3.3 70B)
- **Design Engine**: Jinja2 + Premium CSS3
- **Rendering**: Playwright (Headless Chromium)
- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism), JavaScript (Particle Systems)

## 🚀 Getting Started

### 1. Prerequisites
- Python 3.10+
- A Groq API Key ([Get one here](https://console.groq.com/))

### 2. Installation

Clone the repository and install dependencies:
```bash
pip install -r requirements.txt
python -m playwright install chromium
```

### 3. Configuration
Create a `.env` file in the root directory:
```env
GROQ_API_KEY=your_actual_api_key_here
```

### 4. Run the Agent
```bash
python main.py
```
Visit `http://localhost:8000` to start creating!
